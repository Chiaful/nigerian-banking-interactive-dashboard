## 📌 Overview

The **Nigeria Banking Industry Dashboard** is an interactive, scenario-driven strategic intelligence tool that projects the trajectory of Nigeria's banking sector from **2025 to 2035**.

Designed for executive decision-makers, board presentations, regulatory bodies, and development finance institutions, it translates complex sector data into clear, actionable visual intelligence.

---

## ✨ Features

- **5 Scenario Buttons** — instantly model Base Case, Fintech Surge, Regulatory Reform, Rural Inclusion Push, and Macro Headwinds
- **8 Interactive Sliders** — adjust credit growth, deposit mobilisation, NIM, NPL ratio, financial inclusion, digital adoption, agent network growth, and SME credit access
- **4 Live KPI Cards** — Total Assets 2035, Banked Adults, Industry ROE, Digital Transaction Share
- **3 Chart Views** — Asset & Credit Growth, Financial Inclusion, and Profitability (NII + ROE)
- **Real-time projections** — all charts and KPIs update instantly as variables are adjusted
- **Board-ready design** — clean, professional layout suitable for C-suite and boardroom presentation
